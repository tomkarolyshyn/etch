# etch
Python based CLI for accelerating embedded processing.


## Installation

### OS Support
While generic OS is envisioned and provisioned for, current testing is only being performed on Ubuntu 24.04.

### Prerequistes
Apt / system library installs
1.  build-essential
2.  LLVM via ()

### Etch Instalation.
1.  Create a venv
2.  pip install etch
3.  etch setup
4.  etch --version
